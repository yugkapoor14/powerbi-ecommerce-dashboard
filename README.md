# Ecommerce Sales Dashboard (Power BI)

## Project Overview

This project presents an **Ecommerce Sales Dashboard** created using **Microsoft Power BI**.
The dashboard analyzes ecommerce sales data and provides insights into revenue, orders, cancellations, and sales performance across different products, categories, cities, and time periods.

The purpose of this project is to demonstrate **data visualization and business intelligence skills using Power BI**.

---

## Tools & Technologies Used

* Microsoft Power BI
* Microsoft Excel / CSV Dataset
* Data Visualization
* Data Analysis

---

## Dashboard Features

The dashboard includes the following key metrics and insights:

* **Total Revenue**
* **Average Order Value (AOV)**
* **Cancellation Rate**
* **Lost Revenue due to Cancellations**
* **Total Orders**

---

## Data Visualizations

The dashboard contains multiple visualizations such as:

* **Revenue by Product Name** – Shows top performing products
* **Revenue by Category** – Displays which product categories generate the most revenue
* **Revenue by City** – Identifies cities with highest sales
* **Revenue by Quarter** – Shows sales trend over time

---

## Interactive Filters

Users can interact with the dashboard using:

* **Category Filter**
* **Date Range Slicer**

These filters allow users to analyze the data dynamically.
